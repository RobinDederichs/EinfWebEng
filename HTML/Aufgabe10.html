Aufgabe 10.1 - Static classes in TypeScript

Antwort: In TypeScript kann eine statische Klasse mit dem Schlüsselwort "static" definiert werden. Statische Klassen können nur über ihren Namen direkt angesprochen werden und haben keine Instanzen. Alle Methoden und Eigenschaften, die in einer statischen Klasse definiert sind, sind ebenfalls statisch und können ohne Instanzierung direkt über den Klassennamen aufgerufen werden.
Im Gegensatz dazu sind in objekt-orientierten Sprachen wie Java und C# alle Klassen instanzierbar und müssen erst instanziiert werden, bevor man auf ihre Methoden und Eigenschaften zugreifen kann.


Aufgabe 10.2 - Fehler finden mit TypeScript

Gegeben sei folgendes fehlerbehaftete JavaScript-Programm. Ergänzen Sie dieses um möglichst viele Typisierungen und übersetzen Sie es in ein TypeScript-Programm.

const add = (x,y) => x+y;
const equals = (x,y) => x===y;

console.log( add(1,2) );
console.log( add(true, true ) );
console.log( add(true, false ) );
var x = y = 1;
console.log( add(equals(x,y), equals(y,x)) );

Was ist der Fehler in dem gegebenen JavaScript-Code? Welche Fehler meldet TypeScript? Geben Sie weitere Beispiele an, wie TypeScript bei der Fehlersuche helfen kann.


Antwort: In dem JS Programm lassen sich folgende Fehler ausmachen:

  • Die Semantik der add-Funktion lässt sich nicht auf boolean-Werte anwenden.
  • In Zeile 7 wird y als globale Variable genutzt, was in der Regel nicht erwünscht ist.
  • In Zeile 8 werden die Ergebnisse der equals-Funktion als Parameter für add verwendet, was ebenfalls nicht der Semantik von add entspricht.
Durch hinzufügen folgender Typisierungen und Änderungen in TypeScript: 
const add = (x: number, y: number): number => x + y;
  const equals = (x: any, y: any): boolean => x === y;

  console.log( add(1, 2) );
  console.log( add(true, true) );
  console.log( add(true, false) );

  let x: number = 1;
  let y: number = 1;
  console.log( add(equals(x, y), equals(y, x)) );
werden die Aufrufe von add mit boolschen Werten und dem geschachtelten equals als compile Fehler angezeigt.
